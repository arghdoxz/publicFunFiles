## AIC8800 WiFi+BT Setup — Fresh Armbian Install (Orange Pi 5, kernel 6.1.x-vendor-rk35xx)

---

### Clean Slate

```bash
sudo dkms remove aic8800-usb/4.0+git20250410.b99ca8b6-1 --all 2>/dev/null || true
sudo apt remove --purge aic8800-usb-dkms aic8800-firmware -y 2>/dev/null || true
sudo rm -rf /usr/src/aic8800-usb-4.0+git20250410.b99ca8b6-1
sudo rm -rf /var/lib/dkms/aic8800-usb
sudo rm -f /lib/modules/$(uname -r)/updates/dkms/aic_load_fw_usb.ko
sudo rm -f /lib/modules/$(uname -r)/updates/dkms/aic8800_fdrv_usb.ko
sudo rm -f /lib/modules/$(uname -r)/updates/dkms/aic_btusb_usb.ko
```

---

### Step 1: System Packages

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install -y build-essential git usb-modeswitch linux-headers-vendor-rk35xx network-manager
```

---

### Step 2: Clone Driver & Build

```bash
cd /tmp
git clone --branch bluetooth --depth 1 https://github.com/shenmintao/aic8800d80.git aic8800d80-bt
cd aic8800d80-bt/drivers/aic8800
make
```

---

### Step 3: Install Firmware & Driver

```bash
# Install firmware
sudo cp -r /tmp/aic8800d80-bt/fw/aic8800D80 /lib/firmware/

# Create symlink so driver finds firmware at expected path
sudo ln -sf /lib/firmware/aic8800D80 /lib/firmware/aic8800_fw/USB/aic8800D80 2>/dev/null || true

# Install udev rules
sudo cp /tmp/aic8800d80-bt/aic.rules /lib/udev/rules.d/

# Install kernel modules manually
sudo mkdir -p /lib/modules/$(uname -r)/updates/dkms
sudo cp /tmp/aic8800d80-bt/drivers/aic8800/aic_load_fw/aic_load_fw.ko \
  /lib/modules/$(uname -r)/updates/dkms/aic_load_fw_usb.ko
sudo cp /tmp/aic8800d80-bt/drivers/aic8800/aic8800_fdrv/aic8800_fdrv.ko \
  /lib/modules/$(uname -r)/updates/dkms/aic8800_fdrv_usb.ko
sudo depmod -a
```

---

### Step 4: Init Script

```bash
sudo tee /usr/local/bin/aic8800-init.sh << 'EOF'
#!/bin/bash
sleep 5
if lsusb | grep -q "a69c:5721"; then
    usb_modeswitch -v 0xa69c -p 0x5721 -KQ
    sleep 6
fi
modprobe aic_load_fw_usb
modprobe aic8800_fdrv_usb
sleep 2
echo "368b 8d81" > /sys/bus/usb/drivers/aic8800_fdrv/new_id 2>/dev/null || true
EOF
sudo chmod +x /usr/local/bin/aic8800-init.sh
```

---

### Step 5: Systemd Service

```bash
sudo tee /etc/systemd/system/aic8800.service << 'EOF'
[Unit]
Description=AIC8800 WiFi/BT init
After=network-pre.target
Wants=network-pre.target

[Service]
Type=oneshot
RemainAfterExit=yes
ExecStart=/usr/local/bin/aic8800-init.sh
TimeoutStartSec=60

[Install]
WantedBy=multi-user.target
EOF
sudo systemctl daemon-reload
sudo systemctl enable aic8800.service
```

---

### Step 6: Udev & Modprobe

```bash
sudo tee /etc/udev/rules.d/50-aic8800.rules << 'EOF'
ACTION=="add", SUBSYSTEM=="usb", ATTR{idVendor}=="a69c", ATTR{idProduct}=="5721", RUN+="/usr/local/bin/aic8800-init.sh"
ACTION=="add", SUBSYSTEM=="usb", ATTR{idVendor}=="368b", ATTR{idProduct}=="8d81", RUN+="/bin/sh -c 'sleep 3 && echo 368b 8d81 > /sys/bus/usb/drivers/aic8800_fdrv/new_id'"
EOF

echo 'options usb-storage quirks=a69c:5721:i' | sudo tee /etc/modprobe.d/usb-storage-aic.conf

# Remove any leftover blacklists
sudo rm -f /etc/modprobe.d/blacklist-btusb.conf
sudo rm -f /etc/modprobe.d/blacklist-aic-btusb.conf
sudo rm -f /etc/modprobe.d/blacklist-aic-sdio.conf

sudo udevadm control --reload-rules
sudo depmod -a
```

---

### Step 7: Boot Time Optimization

```bash
# Disable systemd-rfkill (saves ~11s)
sudo systemctl mask systemd-rfkill.service
sudo systemctl mask systemd-rfkill.socket

# Disable NetworkManager-wait-online (not needed for desktop)
sudo systemctl mask NetworkManager-wait-online.service

# Disable unused services
sudo systemctl disable rpcbind.service
sudo systemctl mask rpc-statd-notify.service

sudo systemctl daemon-reload
```

---

### Step 8: Reboot & Verify

```bash
sudo reboot
```

After reboot:

```bash
# WiFi
ip link show | grep wl
sudo nmcli dev wifi connect "YourSSID" password "YourPassword"
sudo ping -c 3 8.8.8.8

# Bluetooth
hciconfig -a
bluetoothctl show
```

---

**Expected results:**
- WiFi: `wlx...` interface appears, connects to SSID, ping succeeds
- BT: `hciconfig` shows `UP RUNNING` with a real MAC address, `bluetoothctl show` shows controller

**If kernel is ever updated**, rebuild the driver:
```bash
cd /tmp/aic8800d80-bt/drivers/aic8800
make clean && make
sudo cp aic_load_fw/aic_load_fw.ko /lib/modules/$(uname -r)/updates/dkms/aic_load_fw_usb.ko
sudo cp aic8800_fdrv/aic8800_fdrv.ko /lib/modules/$(uname -r)/updates/dkms/aic8800_fdrv_usb.ko
sudo depmod -a && sudo reboot
```
